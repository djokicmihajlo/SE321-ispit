# Auto Parts Shop

Full-stack aplikacija za prodavnicu auto delova prema `specs.md`.

## Lokalna baza

Projekat koristi MAMP MySQL:

```text
Host: localhost
Port: 8889
Username: root
Password: root
```

Planirane baze:

```text
auto_parts_shop_dev
auto_parts_shop_test
```

## Pokretanje

Backend:

```bash
npm run backend
```

Frontend:

```bash
npm run frontend
```

## Provere

```bash
npm run test:backend
npm run test:frontend
npm run build
```
